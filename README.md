# OrzuMall Railway Bot

Bu loyiha Railway va GitHub uchun tayyor.

## Repo ichida bo‘lishi kerak
- bot.py
- requirements.txt
- Procfile

## Railway ulash
1. GitHub'da yangi repo oching.
2. ZIP ichidagi fayllarni repo root qismiga yuklang.
3. Railway'da shu repo'ni ulang.
4. Variables bo‘limiga `.env.example` dagi qiymatlarni kiriting.
5. Botni kanalingizga admin qiling.
6. Deploy qiling.
